Receipt Printer

Personal Project

Type Software, Embedded Programming
Status Ongoing
Year 2026 – Present
Tools VSCode / PlatformIO, Python
[ Image ]

Overview

A personal project built around a thermal receipt printer and a microcontroller — a platform for printing ephemeral, physical output from software. Python handles formatting and API calls on the host side; firmware on the microcontroller reads from the serial monitor and drives the printer. Four distinct modes have been built so far, each treating the receipt as a different kind of output medium.

Modes

Album Mode

Uses the Spotify API to pull album data and generate a formatted receipt — tracklist, metadata, and a space for personal notes. Prompts the user for their take on the album, then auto-formats the responses and converts the layout to serial commands uploaded to the microcontroller for printing.

Poem Mode

Takes any poem as input and reformats it to receipt proportions — narrow columns, deliberate line breaks — then sends it to the printer. The receipt as broadside.

Typewriter Mode

Prints a line automatically whenever a newline is written to a watched file, turning the printer into a slow, physical typewriter. Write in the file; the receipt unspools in real time.

Dictionary Mode

Calls a dictionary API and prints the definition of any word on demand — a physical lookup slip.

Next

A custom enclosure to house the printer cleanly, and an automatic cutter so receipts separate on their own as they feed out. Integration with calender to create to do lists.
