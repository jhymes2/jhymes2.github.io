Title: Receipt Printer Project
Status: Ongoing
Year: 2026 - Present
Stack: Software, Embedded Programming
Tools: VSCode / PlatformIO, Python

Created a reciept printer program for a few different purposes

Album mode:
uses spodify API to pull album data and create a formatted reciept with my opinions on the album. Prompts user their opiniosna nd automatically formats reciept. then converts this fomat to code that is uploaded to a microcontroller connected to a reciept printer. Software on the microcontroller takes readings from the serial moniter to print reciept.

Poem mode:
Takes a poem and formats to reciept style and uploads to printer

Typewriter mode:
Prints line automatically after a newline is called in a file, allowing the printer to act as a small typewriter

Dictionary mode:
Calls from an API to print definitions of words

Future idertations: create sleek case and automatic cutting of reciepts as they come out of the printer
